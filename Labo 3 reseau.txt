- ipconfig (verifier les paramètres réseau de la machine)
- ipconfig /all (verifier plus des paramètres réseaux de la machine)
- route print (affichage table de routage)
- netsh interface show interface (verifier nom et status des interfaces)
- netsh interface ip set address "Ethernet" static <IPV4_address> <netmask> [<Default_Gateway>] (configurer une adresse statique sur une interface)
- netsh interface ip add address "Ethernet" <IPV4_address> <netmask> (ajouter une adresse supplémentaire)
- netsh interface ip delete address "Ethernet" <IPV4_address> (supprimer des adresses)
- netsh interface ip set address "Ethernet" dhcp (configurer une adresse dynamique sur une interface)
- netsh interface ip set dns "Ethernet" static <IPV4_address> (ajouter un serveur dns)
- netsh interface set interface "Ethernet" enable (activer une carte réseau)
- netsh interface set interface "Ethernet" disable (désactiver une carte réseau)
- route add [-p] 0.0.0.0 mask 0.0.0.0 <IPV4_address> (ajout d'une route par défaut)
- route delete 0.0.0.0 (suppression route par défaut)
